     The first aim is to provide comprehensive bibliographical support for six aspects of cultural history as they relate to ancient Mesopotamia. In each webasite, one will find in section I. THE ARGUMENT a discursive approach to these topics, and in section II. THE RECORD a set of data in a form that is consonant with a standard annotated bibliography.
     One will find a detailed description of this aim in the SUBSTANTIVE sction of this webbsite as shown in the side bar to the left.

     Undergirding the substantive treatment of the data is a second, methodological, aim: I intend to show how a browser based approach, as available in a website, provides a good tool for combining a data set with an argument based narrative. This is achieved by interlacing the sequential texts as given in section I. THE ARGUMENT with the data found in section II. THE RECORD, and viceversa.
     One will find a detailed description of this aim in the DIGITAL sction of this webbsite as shown in the side bar to the left.
