Perspectives

     Corresponding to the first aim (a comprehensive bibliographical support, one will find in section I. THE ARGUMENT a discursive approach to these topics, and in section II. THE RECORD a set of data in a form that is consonant with a standard annotated bibliography.
     Corresponding to the second, methodological, aim, I intend to show how a browser based approach, as available in a website, provides a good tool for combining a data set with an argument based narrative. This is achieved by interlacing the sequential texts as given in section I. THE ARGUMENT with the data found in section II. THE RECORD, and viceversa.
