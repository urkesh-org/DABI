Overview

Giorgio Buccellati – March 2021

     This website serves as an introductory hub to the cluster of six websites listed on the home page.
     Part 1. CONCEPTS provides a brief review of the major theoretical issues the cluster aims to bring into focus, in particular with regard to the nature of digitality.
     Part 2. COROLLARIES develops some points that relate to the implementation of the project.
     Part 3. SUBSTANTIVE describes how the websites can be used as tools for bibliographical research.
     Part 4. METHODOLOGICAL describes how the various components of the websites implement concretely the notion of digital discourse, which is the primary methodological goal of the cluster.
     Part 5. MECHANISMS presents the infrastructure that is in place to keep the system functioning.
