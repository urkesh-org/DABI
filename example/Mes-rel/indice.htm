11. Indices

     Several indices are produced by a program specifically designed and written for our project.
     Each new version of the website will obviously have an updated set of indices.

     In this section one will find three indices. The first is an extensive topical index, that links individual topics to all the sections of the program.
     The other two indices relate to the bibliography. The first lists the entries in chronological order, while the second gives a list by individual author.
     The indices pertaining to entries produced by contributors to the website are given in section 13. Authorship, under each contributor's name in the left side bar.

     There is at this point no special search option for the entire website.
